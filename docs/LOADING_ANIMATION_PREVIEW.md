## **Animation Features**

### **1. Icon Animations:**
- **Pulsing Background** - 3 concentric circles expanding/contracting
- **Bouncing Icon** - Step-specific icon (Languages, Brain, Sparkles, Activity) bouncing
- **Color-Coded** - Each step has unique color scheme:
  - Stage 1 (Language): Blue (`text-blue-600`, `bg-blue-50`)
  - Stage 2 (NER): Purple (`text-purple-600`, `bg-purple-50`)
  - Stage 3 (RAG): Yellow (`text-yellow-600`, `bg-yellow-50`)
  - Stage 4 (LLM): Green (`text-green-600`, `bg-green-50`)
  - Stage 5 (Translation): Indigo (`text-indigo-600`, `bg-indigo-50`)

### **2. Progress Bar:**
- **Gradient Fill** - Blue → Purple → Indigo gradient
- **Shimmer Effect** - White highlight moving across bar
- **Smooth Transitions** - CSS `transition-all duration-300`

### **3. Step Indicators:**
- **Circular Badges** - 5 circles showing all steps
- **State Changes:**
  - **Completed:** Green background + white checkmark
  - **Active:** Colored background + scale-110 + ring effect
  - **Pending:** Gray background + scale-90
- **Icons Rotate** - Each step shows relevant icon

### **4. Text Updates:**
- **Main Text** - Bold 2xl font showing current action
- **Detail Text** - Small gray text explaining what's happening
- **Time Counter** - Real-time countdown (19s → 0s)
- **Medical Facts** - Educational tips changing with each step

## **CSS Animations Used**

```css
/* Pulsing circles */
@keyframes ping {
  75%, 100% {
    transform: scale(2);
    opacity: 0;
  }
}

/* Bouncing icon */
@keyframes bounce {
  0%, 100% {
    transform: translateY(-25%);
    animation-timing-function: cubic-bezier(0.8, 0, 1, 1);
  }
  50% {
    transform: translateY(0);
    animation-timing-function: cubic-bezier(0, 0, 0.2, 1);
  }
}

/* Shimmer effect on progress bar */
@keyframes shimmer {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(100%);
  }
}

/* Fade in animation */
@keyframes fade-in {
  from {
    opacity: 0;
  }
  to {
    opacity: 1;
  }
}

/* Slide in from bottom */
@keyframes slide-in-from-bottom {
  from {
    transform: translateY(20px);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}
```
